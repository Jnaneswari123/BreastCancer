# Breast Cancer Prediction System

This project is a Machine Learning based web application that predicts breast cancer using tumor feature inputs. The model is trained using the Breast Cancer Wisconsin dataset and deployed using Streamlit.

## Features
- Real-time prediction
- User-friendly interface
- Machine Learning model integration
- Streamlit web deployment
- Accurate tumor classification

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Pickle
